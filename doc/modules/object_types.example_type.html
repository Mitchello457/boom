keydown (this, key)
Custom event handler (keydown) In the ‘init’ event, we subscribed to the ‘keydown’ event. So, whenever a key is pressed we will receive a call to ‘keydown’ with the key passed as the first arg. (information on the ‘keydown’ event can be found in /event.lua )

Parameters:

  • this
  • key Just write the key to stdout.
