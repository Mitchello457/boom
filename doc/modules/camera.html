Module camera

Camera subsystem.

The game will look “crispest” if we scale pixels properly (no half-pixels ever rendered). So, we apply the camera using only a scale and a translate to center on the screen.

Functions

center (x, y) Center the game camera on a point.
setshake (Time) Set the camera shake time.
setscale (scale) Set the size (in display pixels) for 1 game pixel.
get_bounds () Get the camera bounding rectangle.
get_center () Get the camera center.
apply () Apply the camera settings to the graphics context.
unapply () Unapply the camera settings in the graphics context.
set_focus_x (x) Set the camera focus X.
set_focus_flip (enabled) Enable or disable mirroring of the focus box.
set_focus_y (y) Set the camera focus Y.
render_debug () Render camera debug information.
