Functions

init (tiled_obj, tilesets_obj, physics_world)
Initialize a tile layer. Prepares a tile layer for rendering and collisions.

Parameters:

  • tiled_obj Tile layer table from map file.
  • tilesets_obj Tilesets table from map file.
  • physics_world Physics world to use for solid layers.

Returns:

    The initialized tile layer.
unload (layer)
Unloads a tile layer. Destroys any physics objects in solid layers.

Parameters:

  • layer Layer to unload.
render (layer)
Render a tile layer.

Parameters:

  • layer Tile layer to render.
aabb (layer, rect)
Check for AABB collisions against a layer. If an edge of the rectangle overlaps a tile edge is it said to NOT be colliding.

Parameters:

  • layer Tile layer to test.
  • rect Rectangle to test, should have numeric fields x, y, w, h.

Returns:

  1. True if rect intersects with a non-zero tile ID in layer.
  2. Rectangle of the collided tile. Will have numeric fields x, y, w, h.

Or

    False if rect does not intersect with any nonzero tiles in layer.
