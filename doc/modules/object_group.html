init (tiled_obj)
Initialize an object group from a Tiled map.

Parameters:

  • tiled_obj Lua object parsed from map file.

Returns:

    The object group interface.

Remove dead objects from a group.

Parameters:

  • group Object group to clean
unload (group)
Unload and release all resources from a group. This will destroy every object in the group.

Parameters:

  • group Group to unload
call (group, func, ...)
Call a handler on each object (if present)

Parameters:

  • group Group to operate on
  • func Handler name to call
  • ... Arguments to pass to handler.
find (group, name)
Search for an object by name. Returns nil if the object is not found.

Parameters:

  • group Group to search in
  • name Object name to search for

Returns:

    The object if found, otherwise nil
find_type (group, type_name)
Collect all objects of a certain type. Returns an array of all matching objects in this layer.

Parameters:

  • group Group to search in
  • type_name Type name to search for

Returns:

    An array of objects of type type_name.
create_object (group, type_name, initial)
Create a new object and add it to the group.

Parameters:

  • group Object group to add to.
  • type_name Type name of new object.
  • initial Initial state of new object.

Returns:

    The newly created object, or nil if an error occurs
