Functions

read_map (name)
Read a map from the disk. Attempts to load name.lua from the map prefix (assets/maps).

Parameters:

  • name Map name to load.

Returns:

    The map table, or nil if an error occurs.
read_texture (name)
Read a texture from the disk. Attempts to load a sprite from the sprite prefix (assets/sprites). Any path prefixes are stripped off the beginning of the texture name.

Parameters:

  • name Texture name to load.

Returns:

    The loaded texture, or nil if an error occurs.
