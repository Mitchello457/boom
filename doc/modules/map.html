Functions

load (name)
Immediately load and initialize a map. Loads map from the disk, and unloads any current map.

Parameters:

  • name Map to load.
unload ()
Unload the current map, if there is one loaded. Destroys all resources and content loaded in the current map.
get_physics_world ()
Get the map physics world.

Returns:

    The active physics world or nil if no map.
update (dt)
Update all objects in the map y dt seconds. Also updates the map’s physics world. Any objects marked for destruction are destroyed afterward.

Parameters:

  • dt Ignore if no map loaded.
render ()
Render all objects in the map.
find_layer (name)
Find a layer by name.

Parameters:

  • name Layer name to search for.

Returns:

    First matching layer with name or nil if not found.
find_object (name)
Find an object by name. Searches each object group for an object named . Returns the first valid result.

Parameters:

  • name
aabb_tile (rect)
Check for AABB collisions on all solid tile layers.

Parameters:

  • rect Rectangle to test. Should have numeric fields x, y, w, h.

Returns:

  1. True if rect intersects with a non-zero tile ID in layer.
  2. Rectangle of the collided tile. Will have numeric fields x, y, w, h.

Or

    False if rect does not intersect with any nonzero tiles in layer.
