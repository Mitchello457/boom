Functions

create (tex_path, frame_w, frame_h, duration)
Create a new sprite.

Parameters:

  • tex_path Texture name. Passed to fs.read_texture.
  • frame_w Width of a single frame.
  • frame_h Height of a single frame.
  • duration Delay between frames (seconds).

Returns:

    The sprite object.
frame (spr)
Get the current sprite frame texture.

Parameters:

  • spr Sprite to query.

Returns:

    The current texture.
update (self, dt)
Update a sprite by dt seconds.

Parameters:

  • self Sprite to update.
  • dt Seconds to update by.
play (self)
Play or resume a sprite. Will start the sprite from the beginning if it is already at the end.

Parameters:

  • self Sprite to play.
pause (self)
Pause a sprite.

Parameters:

  • self Sprite to pause.
stop (self)
Stop a sprite. Pauses the sprite and moves the frame to the start.

Parameters:

  • self Sprite to stop.
render (self, x, y, angle, flipped)
Render a sprite.

Parameters:

  • self Sprite to render.
  • x X coord for the top-left corner.
  • y Y coord for the top-left corner.
  • angle Rotation about the center.
  • flipped If true, horizontally flip the sprite.
